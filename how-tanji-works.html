Tanji is a platform that allows households to easily manage their waste by providing them with the ability to liase with nearby garbage collectors to collect their waste.

To access Tanji's services, one is requested to sign up via the sign up page. Here, the user provides their identity information as well as their place of residence. When the account creation is successful, the user is required to login. The login process verifies the login and once successful, the user is taken to their dashboard.

At the dashboard, the user is provided with an easy-to-use interface. At the menu, the user can select a range of options. To request for garbage collection, the user selects the Request Collection option at the top menu. This redirects to the request collection page where the user selects from the provided map their exact location. Upon clicking the request Collection button, the user is provided with a list of the available garbage collection companies. The user selects their most preferred and proceeds to confirm their order.
